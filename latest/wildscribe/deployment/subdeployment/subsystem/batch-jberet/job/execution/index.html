  6. execution

The execution information for the job with the value of the path being the execution id.

Runtime only resource This resource is runtime only and doesn't have any persistent configuration.

Operations (2)

  • restart-job Restarts a batch job. Only jobs in a STOPPED or FAILED state can be restarted. This must also be the most recent job execution.

  • stop-job Stops a running batch job.
