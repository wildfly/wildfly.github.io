  4. subsystem=batch-jberet

Information about the batch subsystem for the deployment.

Runtime only resource This resource is runtime only and doesn't have any persistent configuration.

Children (1)

  • job Information about a specific batch job.

Attributes (1)

  • job-xml-names A list of job XML job descriptors found for the deployment.
    • Attribute Value
    Type LIST
    Nillable false
    Expressions Allowed false
    Storage runtime
    Access Type read-only

Operations (3)

  • restart-job Restarts a batch job. Only jobs in a STOPPED or FAILED state can be restarted.
    Request Parameter Type Required Expressions Allowed Default value Description
    execution-id LONG true false The execution id of the job to restart. This must be the most recent job execution id.
    properties OBJECT false false Optional properties to use when restarting the batch job.

    Reply properties

    type
    LONG
  • start-job Starts a batch job.
    Request Parameter Type Required Expressions Allowed Default value Description
    job-xml-name STRING true false The name of the job XML descriptor to use when starting the job.
    properties OBJECT false false Optional properties to use when starting the batch job.

    Reply properties

    type
    LONG
  • stop-job Stops a running batch job.
    Request Parameter Type Required Expressions Allowed Default value Description
    execution-id LONG true false The execution id of the job to be stopped.
