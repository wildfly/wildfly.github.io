Information about a specific batch job.

Runtime only resource This resource is runtime only and doesn't have any persistent configuration.

Children (1)

  • execution The execution information for the job with the value of the path being the execution id.

Attributes (3)

  • instance-count The number of instances for the job.
    • Attribute Value
    Type INT
    Nillable false
    Expressions Allowed false
    Storage runtime
    Access Type read-only
  • job-xml-names A list of job XML job descriptors found that describe this job.
    • Attribute Value
    Type LIST
    Nillable false
    Expressions Allowed false
    Storage runtime
    Access Type read-only
  • running-executions The number of currently running executions for the job.
    • Attribute Value
    Type INT
    Nillable false
    Expressions Allowed false
    Storage runtime
    Access Type read-only
