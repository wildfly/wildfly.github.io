  2. deployment-overlay

A deployment overlay

Children (2)

  • content The content of the deployment overlay
  • deployment The deployment that this deployment overlay is linked to

Operations (3)

  • add Adds a deployment overlay
  • redeploy-links Redeploys the deployments affected by this overlay.
    Request Parameter Type Required Expressions Allowed Default value Description
    deployments LIST false false List of deployment runtime names to be redeployed. If this list is empty or 'undefined' then all affected deployments will be redeployed.
  • remove Removes a deployment overlay
