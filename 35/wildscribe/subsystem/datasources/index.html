  2. subsystem=datasources

The data-sources subsystem, used to declare JDBC data-sources

Children (3)

  • data-source A JDBC data-source configuration
    • ExampleDS A JDBC data-source configuration
  • jdbc-driver Service that makes a JDBC driver available for use in the runtime
  • xa-data-source A JDBC XA data-source configuration

Attributes (1)

  • installed-drivers List of JDBC drivers that have been installed in the runtime
    • Attribute Value
    Type LIST
    Nillable false
    Expressions Allowed false
    Storage runtime
    Access Type read-only
    Stability default

Operations (4)

  • add Adds the data-sources subsystem
